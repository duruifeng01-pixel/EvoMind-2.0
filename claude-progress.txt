=== 技术栈 ===

Android端:
- Kotlin + Jetpack Compose
- MVVM + Clean Architecture
- Room数据库
- Hilt依赖注入
- 百度OCR/语音SDK
- ONNX Runtime（本地AI）

后端:
- Spring Boot 3.x + Java 17
- MySQL 8.0 + Redis
- 百度文心一言API
- 微信支付/支付宝
