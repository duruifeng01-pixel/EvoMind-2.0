## 📋 详细任务分解

### feat_073: 密码登录与忘记密码功能

**后端任务**:
- [ ] User实体添加password字段(BCrypt加密)
- [ ] POST /auth/login-by-password 密码登录接口
- [ ] POST /auth/forgot-password 忘记密码申请
- [ ] POST /auth/reset-password 验证码重置密码
- [ ] 密码强度校验规则(8位以上，含大小写+数字)

**Android任务**:
- [ ] 密码登录页面UI
- [ ] 忘记密码流程(手机号验证+新密码设置)
- [ ] 密码输入安全键盘
- [ ] 登录状态持久化

**数据库迁移**:
```sql
ALTER TABLE users ADD COLUMN password VARCHAR(255);
ALTER TABLE users ADD COLUMN password_updated_at DATETIME;
```

### feat_076: 算力成本统计与透明化展示

**后端任务**:
- [ ] ComputingCostRecord实体(成本记录)
- [ ] CostStatisticsService(成本统计)
- [ ] 成本指标：信息源数/冲突标记/token消耗/对话轮数
- [ ] 成本单价配置表
- [ ] GET /computing-cost/stats 算力成本统计
- [ ] GET /computing-cost/estimate 预估订阅费
- [ ] 定时任务：每10分钟更新成本

**Android任务**:
- [ ] 算力成本展示页面
- [ ] 成本明细列表
- [ ] 订阅费计算公式展示

**定价规则**:
```
订阅费 = 预估算力成本 × 2
显示：算力成本(80%) + 运营成本(20%)
```
